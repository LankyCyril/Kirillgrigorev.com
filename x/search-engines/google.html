Install the Better Google search template for Firefox

Depending on the Firefox flavor, either:
click the magnifying glass icon in the Firefox search bar; or
click the page actions icon ("...") in the location bar; or
right-click anywhere in the location bar.

Then select the option to either "Add Better Google" or "Add Search Engine."
